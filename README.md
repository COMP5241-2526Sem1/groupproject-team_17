# COMP5241-Project-Sem1-Grp17

This repository contains the group project for COMP5241 Semester 1, Group 17.

## Project Overview
This project aims to develop a comprehensive solution for Interaction between teachers and students. The main objectives include:
- Interactive learning activities
- Real-time feedback and assessment

## Technologies Used
- Frontend: Next.js React
    - MUI for UI components
    - State Management: Redux
    - Routing: App Router
    - HTTP Client: Axios
- Backend: C# ASP.NET Core
    - Authentication: Auth0 with JWT
    - API Documentation: Swagger
    - ORM: Entity Framework Core, MySQL
    


- Database: MySQL

- Version Control: Git and GitHub

## Project Structure
- `frontend/`: Contains all frontend related code and assets.
- `backend/`: Contains all backend related code and configurations.
